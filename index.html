Projects

Campaign Query

Focused on campaign finance data, this project utilizes Amcharts and Highcharts in conjuction with the Federal Election Commission API to display meaningful finance data in a clean, streamlined manner. Created as my second project for web development bootcamp, this was our first full-stack project. My role was project manager in addition to UX/UI and assisting with API calls. My goal with this project was to use simple, modular CSS as opposed to using a framework. It turned out to be much more difficult than initially anticipated, but I was ultimately able to make it work.

appDoption

A front-end, photos focused application for searching adoptable pets in your area. This application was created with the usability of Instagram in mind and connects with the Petfinder API. This was completed as my first project for web development bootcamp. My role on this project was as project manager and UI/UX design and development. For this project I chose to learn the Materialize CSS framework, which ultimately led me to a much better grasp of responsiveness and CSS gridding to achieve the desired layout.

Virtual Hostel

Built using React, this project allows the user to post reviews of travel experiences and view others reviews and suggested itineraries. This was my third and final web development bootcamp project, and I served as project manager in addition to UI design and several React components. In conjunction with MongoDB, I established hashed and secure user authorization with Passport.js. Additionally I worked on the components and functionality for setting favorites and tying them to user accounts.

About

My name is Amy. I'm a developer, with a love for CSS and all things style sheets. Originally from Tennessee, I've lived in Chicago by way of New Jersey for the last 8 years with my wonderful husband and our two cats.

I started out designing and building websites for friends back in the early 2000s, when in-line scrolling frames were all the rage and we built our designs in Paintshop Pro.Fast forward about 15 years, I've now had a career as a flight attendant and spent a considerable amount of time in specialty coffee . I guess you could say I love traveling and good coffee.

My back end developer husband helped me reignite my passion for web development and design, leading me to enroll at the Northwestern University Web Dev Bootcamp. I've brushed up on HTML and CSS, and stuck my toes in the JavaScript water. I've learned that I love learning new development languages, and building beautiful things. Now I'm excited to see where development can take me.

When I'm not developing or perusing the internet, I enjoy things like playing video games, catching all the Pokemon, and shopping for animation cels to add to my ever growing collection.
