Projects


appdoption

Version 1 | Github
Version 2 | Github

A front-end, photos focused application for searching adoptable pets in your area. This application was created with the usability of Instagram in mind and connects with the Petfinder API, and was created as a way to easily view adoptable pets in the searched area. This was completed as my first project for web development bootcamp. My role on this project was as project manager and UI/UX design and development. For this project I chose to learn the Materialize CSS framework, which ultimately led me gaining much better grasp of responsiveness and CSS gridding to achieve the desired layout.

Upon completion of bootcamp this was the first project I wanted to revisit. I spent additional time improving the UI to be more heavily similar to Instagram, as a large portion of our feedback was that given the goals of the project, the instruction team wanted to see it look and feel more like Instagram. With more time and fewer hard requirements, I was also able to improve upon the useage of the api and add an initial display of random results, so that the user could see what sort of content would be displayed once a search was conducted. Each step of the way I ensured that Appdoption would function just as easily on mobile as it does on desktop. In going back through this project I also worked to remove almost all uses of jQuery to improve performance. The only useage is for a simple ajax call, as the no-CORS API does not work with the .fetch() method. I opted not to route the api via a backend server as the document for Petfinder specifically says to not worry about hiding your API key and that being used purely on the front end is recommended.

IsGravityStillWorking.com

Version 1 | Github

Is Gravity Still Working started as a joke, offhandedly made in a Slack channel of friends. I, having plenty of free time that day, decided to make the joke a reality and purchased IsGravityStillWorking.com. The site is simple, when the page loads a random gif of a cat knocking something over is loaded. The site started with 14 gifs, and a dropbox folder for adding new ones as we found ones we liked. It was simple to write the Javascript, and even simpler to create the design. It reminds me a lot of an original internet treasure called You're the Man Now Dog, which was a community of users that would tie together gifs or static images and soundclips to create jokes. Is Gravity Still Working is simple, but perfectly so and was built exactly as intended.

Campaign Query

Version 1 | Github

Created as our second bootcamp project (and first full stack project), Campaign Query aims to display campaign finance data with easy readability. This project utilizes Amcharts and Highcharts in conjuction with the Federal Election Commission API routed through a backend Express server using node.JS. My role for this project was project manager in addition to UX/UI and assisting with API calls. My goal with this project was to use simple, modular CSS as opposed to using a framework. It turned out to be much more difficult than initially anticipated, but I was ultimately able to make it work and was very happy with the result. Campaign Query received high praise and we were later told it was shared among the administrators because of how interesting our instructors found it.

I plan to reapproach Campaign Query in the coming months so that it can be useful for the 2020 election cycle.

Pikachu Rumble

Version 1 | Github

Pikachu Rumble is a simple JavaScript game made in an afternoon as a simple refresher exercise. I based it's functionality off of a simple Javascript game made early in the bootcamp program I attended, which was called Crystal Collector. The game is simple -- hit points and attack power are all chosen at random, and for each attack from the user the opposing Pokemon counterattacks. The first one to zero wins!
